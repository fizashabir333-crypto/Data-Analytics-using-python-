# Data Analytics Using Python

## Project Overview

This repository contains implementations of various data analytics and machine learning techniques using Python.

## Modules Implemented

1. Data Loading and Exploration (EDA)
2. Data Preprocessing and Cleaning
3. Data Visualization Techniques
4. Statistical Analysis
5. Regression Analysis
6. Classification Techniques
7. Clustering Algorithms
8. Dimensionality Reduction using PCA
9. Time Series Analysis
10. Mini Project: End-to-End Data Analytics Pipeline

## Dataset Information

Some datasets are included in this repository for demonstration purposes.

Note: Large datasets such as the credit card dataset are not uploaded because of file size limitations.

Credit Card Dataset Source:

Kaggle Credit Card Fraud Dataset:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
